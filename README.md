# Spendly 💰

A full-stack personal expense tracker built with ASP.NET Core MVC 8.

## Tech Stack
- ASP.NET Core MVC 8
- Entity Framework Core 8
- ASP.NET Identity (Auth & Roles)
- SQL Server
- Bootstrap 5

## Features
- [ ] User registration & login
- [ ] Add / edit / delete expenses
- [ ] Category management
- [ ] Analytics dashboard with charts
- [ ] CSV export
- [ ] REST API
- [ ] Admin panel
- [ ] Deployed on Azure

## Getting Started
1. Clone the repo
2. Update `appsettings.json` with your connection string
3. Run `Update-Database` in Package Manager Console
4. Press F5
